    Simple and Stylish Image Slider

    SnSiS was experimental project, but now i decided to develop it for newbies. It is easy to use. The good feature of SnSiS is you may customize it as you wish easily.

    How to use

    Load SnSiS plugin after jQuery in your web page. then use $("#wrapper").snsis(); to create events on elements. now you have installed SnSiS plugin. but in following i describe its options that you need for customizing.

    [options]

    fadingSpeed

    Images fade in and fade out, it has set to 400 as default, but you can set integer or set slow or fast string.

    imagesContainer

    The element that consist of images for sliding. it has set to #container as default.

    keyboardArrows

    This plugin get you sliding using keyboard left and right arrows. but you can disable it by giving false to this option.

    fullscreen

    By clicking on images the fullscreen of the activated image will be run as default, but you can disable it by giving false to this option.
